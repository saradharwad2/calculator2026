# Calculator App

A simple and responsive calculator application that performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

## 🚀 Features

* Addition (+)
* Subtraction (-)
* Multiplication (×)
* Division (÷)
* Clear display
* User-friendly interface
* Responsive design

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📂 Project Structure

```text
Calculator/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/calculator.git
```

2. Open the project folder.

3. Run `index.html` in your browser.

## 📖 Usage

* Enter numbers using the calculator buttons.
* Select an arithmetic operator.
* Press `=` to view the result.
* Use `C` or `AC` to clear the display.
